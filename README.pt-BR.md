# AuxiTalk Dashboard

> Dashboard visual oficial do AuxiTalk (plugin ui especial).

> English: [README.md](README.md)  
> Português: este arquivo

## Visão geral

O Dashboard é um plugin `ui` que vem junto do AuxiTalk Core para permitir visualizar e controlar o runtime de forma simples e visual.

Ele sobe um servidor HTTP local e é acessível via navegador.
